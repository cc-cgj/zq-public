函数：就是封装了一段可被重复调用执行的代码块。通过此代码块可以实现大量代码的重复使用。

函数的封装是把一个或者多个功能通过函数的方式封装起来，对外只提供一个简单的函数接口。

函数的参数：形参：函数定义时设置接收调用时传入 实参：函数调用时传入小括号内的真实数据

	函数可以带参数也可以不带参数 声明函数的时候，函数名括号里面的是形参，形参的默认值为 undefined 调用函数的时候，函数名括号里面的是实参多个

参数中间用逗号分隔(形参的个数大于实参个数时，多余的形参为undefined)

------函数的返回值默认值为undefined

	注意：
		1. return 终止函数
		2. return 只能返回一个值（如果return两个值得话，只返回最后一个值）
		3. 可以进行加减乘数除运算，最后返回该结果
		4. 返回可以是一个数组或对象
		5.我们的函数如果有return 则返回的是 return 后面的值，如果函数么有 return 则返回undefined

js中定义一个函数(function)，再给它赋予数值(number)，此时该函数变为number类型了

立即执行函数: ;( function(){} )() 

------break,continue,return的区别：
	break ：结束当前的循环体（如 for、while）
	continue ：跳出本次循环，继续执行下次循环（如 for、while）
	return ：不仅可以退出循环，还能够返回 return 语句中的值，同时还可以结束当前的函数体内的代码

------arguments:用于获取函数调用时所传递的所有实参，展示形式为一个伪数组(Array.protoptype.slice.apply(arguments)转真数组)

	1、具有length属性

	2、按索引方式存储数据

	3、不具有数组的API(push、pop)等方法

函数的两种定义方式
	1、利用函数关键字自定义函数(命名函数)：function 函数名(){}
	2、函数表达式(匿名函数)：var 变量名 = function(){}

	


一、字符串的不可变（内存表现形式）

	概念：

		当重新给字符串变量赋值的时候，变量之前保存的字符串在内存中不会被清空，依然保存在内存中，而之所以字符串的值能被修改，是因为又

	重新开辟新的内存空间存放当前修改的值，然后将值的地址赋予给变量。

二、字符的"索引"方法：

	1、indexOf():字符串中查找给定元素的第一个索引(从左到右)，如果存在返回索引值，没有则返回-1

		（indexOf(所要查找的元素(字符串/数值),第几个开始(数值),结束在第几个(数值可以为负数，包含最后结束的那一个)）

		注意：区分大小写

	2、lastIndexOf():字符串中查找给定元素的最后一个索引(从左到右)，如果存在返回索引值，没有则返回-1

三、根据位置返回字符:

	1、字符串.charAt(index（字符串的索引号）):返回指定位置处的字符

	2、字符串.charCodeAt(index):返回指定位置处字符的ASCII码

	3、字符串.str[index]:返回指定位置处的字符(HTML5)

	注意：字符编码(ASCII)：0(48)、A(65)、a(97)【往后以此类推加1】

四、字符串的操作方法：

	1、字符串名1.concat(字符串名1,字符串名2,...):用于连接两个或多个字符串，以逗号分隔

		(注意：不会改变原字符串且返回一个新的字符串)

	2、substr():可在字符串中抽取从 start 下标开始的指定数目的字符。

	字符串名.substr(起始下标(必需，负数则从右边开始算起),取的字符串数(必须是数值。如果省略了该参数,则从开始位置到结尾的字串))

		(注意：不改变原字符串，返回一个新的字符串)

	3、slice(strat,end)：从start位置开始截取到end位置，end取不到(它们两都是索引号)

		(注意：不改变原字符串，返回一个新的字符串)

	4、substring(strat,end)和slice一样返回一个新的字符串，但不接受负数

	5、字符串.replace(被替换的字符串，要替换为的字符串)方法用于在字符串中用一些字符替换另一些字符。

		（注意：不会改变原字符串且返回的是一个新字符串。）

	6、字符串.split("分割字符")方法用于切分字符串，它可以将字符串切分为数组。

		（注意：不会改变原字符串且返回的是一个新数组。）
		
		如：
			var str = "a,b,c"

			console.log(str.split(","))//[a,b,c]长度为3

			console.log(str.split())//[a,b,c]长度为1

			var str = "abc"

			console.log(str.split(","))//[a,b,c]长度为1	

五、字符串可以通过索引值的方式获取字符但不可以通过索引值的方式修改字符

		var str = "abcdef"

		console.log(str[1])//b

		str[1] = "B"

		console.log(str[1])//b

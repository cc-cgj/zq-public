

	一、数组对象的两种创建方式

		var 数组名 = []	

		var 数组名 = new Array(数组长度/数组的值)

			如：
				var arr= new Array(2) | [].length = 2 //数组的长度为2和2个空的数组元素

				var arr= new Array(1,2,3)//相当于var arr = [1,2,3]


		// var arr = [ 1,2 ]
		// arr.length = 1 // arr:[1]
			

	二、instanceof 可以判断一个对象是否是某个构造函数的实例（new 构造函数()）：对象名/变量名 instanceof 构造函数名
	
		如：
			console.log(null instanceof Object)//false
			console.log(typeof null)//Object
			var arr = []
			var obj = {}
			console.log(arr instanceof Array)//true
			console.log(obj instanceof Object)//true

	三、Array.isArray(数组名)：判断是否为一个数组,值为布尔型; H5新增的方法 ie9以上版本支持

	四、添加删除数组元素的方法：

		1、数组名.push(参数)：向数组末尾添加一个或多个元素，改变原来的数组并返回新的长度(长度减1)

		2、数组名.pop(无参)：删除数组的最后一个元素，改变原数组并返回它删除的元素的值

		3、数组名.unshift(参数)：向数组开头添加一个或多个元素，改变原来的数组并返回新的长度(长度加1)

		4、数组名.shift(无参):删除数组的第一个元素，改变原数组并返回它删除元素的值

	五、数组的排序的方法：

		1、数组名.reverse(无参)：数组排序的颠倒，会改变原来的数组并返回新的数组

		2、数组名.sort()：数组进行排序，会改变原来的数组并返回新的数组

		（注意：sort()默认是按照数值的第一位从小到大排序，然后再判断数值相同的值的第二位进行从小到大排序）

			----使用sort(参数)进行排序，参数里必须是一个回调函数：

				1、升序:(a和b为数组前后相邻的两个元素)

					数组名.sort(function(a,b){

			   			 return a-b				
					})

				2、降序：

					数组名.sort(function(a,b){

			    			return b-a		
					})


			注意：

				如果想按照其他标准进行排序，就需要提供比较函数，该函数要比较两个值，然后返回一个用于说明这两个值的相对顺序的数字。

				比较函数应该具有两个参数 a 和 b，其返回值如下：

					1、若 a 小于 b，在排序后的数组中 a 应该出现在 b 之前。

					2、若 a 大于 b，在排序后的数组中 a 应该出现在 b 之后。
				
	六、数组的索引方法：

		1、数组名.indexOf():在数组中查找给定元素的第一个索引(从左到右)，如果存在返回索引值，没有则返回-1

		（indexOf(所要查找的元素(字符串/数值),第几个开始(数值),结束在第几个(数值可以为负数，包含最后结束的那一个)）

		2、数组名.lastIndexOf():数组中查找给定元素的最后一个索引(从左到右)，如果存在返回索引值，没有则返回-1

	七、数组转换为字符串：

		1、数组名.toString():把数组转换为字符串，逗号分隔每一项并返回一个字符串

		2、数组名.join('分隔符')：数组转换为字符串，分隔符决定时如何分隔每一项，最后返回一个字符串
				
	八、数组的拼接

		数组名1.concat(数组名1,数组名2,...):用于连接两个或多个数组，以逗号分隔，

			(注意：不会改变原数组且返回一个新的数组)
	
	九、数组的截取

		1、数组名.splice(第几个开始(必选)，删除的个数(必选),数组添加的新的值(可选)):方法从数组中添加/删除项目，然后返回被删除的项目。

			(注意：会改变原数组并返回一个新的数组(一般用于数组的删除和替换))

		2、数组名.slice(第几个开始(索引值)，截取的个数(不包括最后截取的那一个)):数组中返回选定的元素。

			（注意：不会改变原数组，返回一个新的数组,若只给定开始位置则表示从开始位置截取到数组的最末）

			//伪数组转真数组

				Array.prototype.slice.apply( arguments )

	十、数组的遍历
		
		foreach():用于调用数组的每个元素，并将元素传递给回调函数。注意: forEach() 对于空数组是不会执行回调函数的。

			语法：arr.forEach(function(item,index){})
			
			参数：	
				item：当前元素，必需

				index：当前元素的索引值，可选

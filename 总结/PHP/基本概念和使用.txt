
----------PHP

一、服务器
		
	概念：能够为计算机提供服务的机器
	
二、服务器软件
		
	概念：使计算机具备提供某种服务能力的应用软件，称为服务器软件。

	常见的服务器分类
		
		1、文件服务器
		2、数据库服务器
		3、邮件服务器
		4、Web(网站)服务器
		
	常见的Web服务器软件
			
		1、Apache (开源)

		2、Nginx


三、HTTP(web)服务器
	
	概念：即网站服务器，主要提供文档(文本、图片、视频、音频)web浏览服务，一般安装Apache、Nginx服务器软件。	

四、客户端和服务器
	
	客户端：通过安装客户端软件，索取服务的计算机

	服务器：通过安装服务器软件，提供服务的计算机

五、软件架构

	------C/S架构 Client/server
		
			概念：客户端/服务器架构。需要安装对应的客户端软件，才能获取服务器的服务。如：QQ、微信
			特点：
				1、需要安装才能使用；
				2、性能高效，使用更加稳定和流畅

			
	------B/S架构 Brower/Server
			
			概念：浏览器/服务器架构。无需安装专门客户端软件，通过浏览器去请求服务器的服务。
			特点：
				1、只需要安装浏览器即可。
				2、性能相对较差，没有那么流畅和稳定。

七、动态网站和静态网站

	------静态网站

			概念：使用浏览器端语言进行编程，网站由静态代码（HTML.CSS,JS等）组成。
	------动态网站

			概念：网页通过服务器的程序(php等)动态生成。用户可以和服务器进行交互
		
				（可以根据用户输入的不同信息，返回不同的运行结果）

八、网络基础

	------IP地址

			127.0.0.1 一般为本地地址。
			192.168 开头为局域网的地址。

	------域名
			
			概念：域名对应的是IP地址，通过DNS解析成IP地址。
		
			特殊的域名：localhost //对应了本地IP

	------DNS服务器
		
			作用：解析域名的。

			计算机解析的优先级：本机的hosts文件、DNS服务器

	------端口

			概念：计算机与外界通讯交流的出入口，每个端口对应不同的服务。
			
			------常见的端口

				3306------Mysql 默认端口
				80------Apache 默认端口
	
	------本地文件hosts

			概念：将一些常用的网址域名与其对应的IP地址建立一个关联的“数据库”。	
			
			作用：当用户在浏览器中输入域名时，系统会首先自动从Hosts文件中寻找对应的IP地址，一旦找到，系统会立即打开对应网页，如果没有
		找到，则系统会再将网址提交DNS 域名解析服务器进行IP地址的解析。
			
			路径：C:\Windows\System32\drivers\etc

九、PHP基础

	------简介

			概念：PHP 是一种用于服务端的脚本语言。是开源的，支持大多数主流数据库，MySQL，oracle，Redis等

			注意：
				1、文件以.php后缀结尾，所有程序包含在 <?php 这里是代码 ?> 避免使用中文目录和中文文件名
				2、php文件无法直接在网页上打开，需要运行在服务器环境当中

	-------编码设置(解决乱码)
		
			1、<meta charset = "utf-8">    <?php ?>
			2、<?php header("content‐Type:text/html;charset=utf‐8");?>

						
	------基本使用规则：

			1、php 中的注释和 js 中的一样
			2、所有的 php 代码都在<?php ?>内
			3、必须要通过服务器的方式, 访问 php 文件, php 代码才会进行解析
			4、服务器会对 php 代码进行解析执行, 并将结果输出给浏览器
			5、php 中必须以 分号 结尾, 最后一个语句, 可以省略

十、变量操作
		
	------创建
		
			声明变量必须以$符号开头	

	------删除	
			
			unset($变量名)
	
	------判断变量是否设置值
			
			概念：检测变量的值是否设置，并且不是 NULL
			语法：isset($变量名)
		
			例：
				$var;
				echo var_dump(isset($var)); // bool(false)
				//var_dump() 函数用于输出变量的相关信息。

			注意：变量如果没赋值, 直接使用会报错

	-------判断变量是否为空

			概念：在PHP中认为变量的值为：""、0、"0"、NULL、FALSE、[]时为空。
			语法：empty($变量名)
			例：
				$var = null;
				var_dump(empty($var)); //bool(false)


十一、字符串连接符

	在php中，+号只有算数的功能，并不能拼串，点(.)才为连接符

十二、单引号和双引号
	
		概念：字符串的定义可以使用单引号，也可以使用双引号

		区别：单引号不能解析变量，双引号可以



	一、链接伪类
	
		link(未访问)：表示的是正常情况下链接的样式。

		visited(已访问)：代表链接访问后的样式，则一但链接被访问，则之后它的样式就会是你所设置的visited样式。

		hover：鼠标移到链接上时添加的特殊样式

		focus：在一个元素获得焦点时生效，用户可以通过键盘或鼠标激活焦点

		active：在一个元素处于激活状态(鼠标在元素上按下还没有松开)时所使用的样式

		(hover理论上任何元素都可以使用，focus多是针对表单，如input，而active多用于链接)
	
		链接(a)伪类的排序：link、visited、hover、active

	二、位置伪类
		
		ele:first-child :选取属于其父元素的首个子元素的指定选择器 

		ele:last-child :选取属于其父元素的最后一个子元素的指定选择器 

		ele:nth-child(n) ： 匹配属于其父元素的第 N 个子元素，不论元素的类型（odd奇数even偶数）或公式

			（n表示 选中所有的 2n表示偶数 2n+1奇数3n等等）
 
		ele:nth-last-child(n) ：选择器匹配属于其元素的第 N 个子元素的每个元素，不论元素的类型，从最后一个子元 素开始计数（倒数）。 

		//n 可以是数字、关键词

	三、目标伪类选择器(CSS3)

		:target目标伪类选择器 :选择器可用于选取当前活动的目标元素

		例

			:target {color: red; font-size: 30px; }
